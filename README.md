# avr-lcdtest

Arduino Uno R3 C++ code for testing Adafruit RGB LCD connected through I2C.

Most of the work has been done through [AVRTools](https://github.com/igormiktor/AVRTools) C++ library.

## How to build

    # build libavrtools.a 
    (cd AVRTools && make clean && make)
    # build and link
    make
